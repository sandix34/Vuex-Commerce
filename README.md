# vuex-commerce

## Project setup
```
npm install
```
## Install [JSON Server](https://github.com/typicode/json-server) 
```
npm install -g json-server
```
### Start JSON Server
```
npm run jsnsrv
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
